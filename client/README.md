# Avocano Front end

[Lit](https://lit.dev/) front-end. 

## Local Dev


Requires node 10 & npm 6 or higher, and a running [api server](../server/README.md#local-dev)

Set API server: 

```
export API_URL=https://api-HASH-REGION.a.run.app/
```

Install, build, and start client. 

```bash
npm i
npm run build
npm run start
```
